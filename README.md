# Horse
На шахматной доске 8х8 стоит шахматный конь. Больше никого на доске нет. Ваша задача - определить, сколько полей бьет этот конь. С клавиатуры вводится положение лошади в шахматной нотации. В этой системе обозначений вертикаль принято обозначать латинской буквой от а до h, а горизонталь - цифрой от 1 до 8. Буква и цифра не разделяются прбиламы! Выведите на экран количество полей, которые бьет лошадь.

  1. Введение - е4, Вывод - 8.
  2. Введение - h8, Вывод - 2.
