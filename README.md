# Horse
На шахматной доске 8х8 стоит шахматный конь. Больше никого на доске нет. Ваша задача - определить, сколько полей бьет этот конь. С клавиатуры вводится положение лошади в шахматной нотации. В этой системе обозначений вертикаль принято обозначать латинской буквой от а до h, а горизонталь - цифрой от 1 до 8. Буква и цифра не разделяются прбиламы! Выведите на экран количество полей, которые бьет лошадь.

  1. Введение - е4, Вывод - 8.
  2. Введение - h8, Вывод - 2.

# Happy tickets
Известно, что билет с шестизначным номером называется счастливым, если сумма первых трех цифр номера равна сумме последних трех цифр номера в точности равна произведению суммы последних трььох цифр на число К. Например билет с номером 576501 есть 3-счастливым, потому что (5 + 7 + 6) = 3 * (5 + 0 + 1). В этой терминологии "просто счастливы" билеты - это 1-счастливые билеты. Ваша задача - по заданному натуральному числу К найти количество К-счастливых билетов среди всех билетов с номерами от 000000 до 999999.

Введение - 1, Вывод - 552552.

# Snake
Нужно таблицу, содержащую R строк и C столбцов, заполнить "змейкой" последовательными двузначными числами, начиная с числа 10. Сначала первый столбец заполняется сверху вниз, второй - вверх, третий снова вниз и так далее ... (см. Пример). Программа считывает с одной строки два натуральных числа R и C. Общее количество заполнена последовательными числами от 10 до R * C +9.

Введение - 3 5
Вывод -
10 15 16 21 22
11 14 17 20 23
12 13 18 19 24
